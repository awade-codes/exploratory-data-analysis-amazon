# Amazon Product Analysis — Exploratory Data Analysis in R

## Overview
Exploratory data analysis of Amazon India product data using R.
Dataset sourced from Kaggle (1,465 observations, 16 variables).

## Tools Used
- R, readxl, psych, ggplot2

## Key Questions Explored
- How are prices and discounts distributed across product categories?
- Does discount percentage correlate with product ratings?

## Key Findings
- Ratings cluster between 3.8–4.5 regardless of discount level
- Discount percentage shows little correlation with product rating
- Category-level analysis reveals pricing strategy differences
